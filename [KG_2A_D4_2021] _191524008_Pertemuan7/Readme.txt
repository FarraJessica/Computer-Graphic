Nama Program                  : Vines
Penulis Program               : Farra Jessica
Deskripsi                     : Vines merupakan program animasi berbentuk Pattern 2D. Kata "Vines" ini
                                berasal dari bahasa Inggris yang dalam bahasa Indonesia berati tanaman
                                merambat.
                                Pattern 2D ini dipilih karena pola nya menarik dan bisa menggunakan
                                shape yang sudah pernah dibuat, yaitu elips.
                                Transformasi yang dipakai adalah translasi, dan rotasi.
Cara Penggunaan Program       : Klik button berbentuk segitiga untuk melakukan eksekusi program.
                                Setelah tampil scene dari program, maka klik button bertuliskan "Play"
                                untuk melihat
                                tampilan animasi dari program Vines.
